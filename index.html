Personal Projects

  • SmartLight: An IoT-based Buck Converter Solution for Household Lighting Control

    • The project, "IoT-controlled SmartLight Buck Converter," is a technologically advanced solution that utilizes the Internet of Things (IoT) to precisely control the brightness of a household light. The team employed the use of a buck converter, a widely utilized circuit in power electronics, to regulate the output voltage and current of the light, thus allowing for seamless adjustment of the light's brightness to full, half, and off states. The team also conducted extensive simulation and modeling of the circuit design using industry-standard MATLAB software to ensure optimal functionality and performance. This enabled thorough testing and debugging of the circuit, resulting in a highly efficient and robust product that meets the highest standards of reliability. Moreover, the team developed a user-friendly Android app that facilitates remote control of the light, enabling convenient and effortless operation of the light from any location. This is a perfect solution for those who want to control the lights remotely and easily. The project is a testament to the team's engineering expertise and their ability to seamlessly integrate IoT technology to enhance the functionality and usability of household lighting. It is a solution that is not only practical but also provides an unprecedented level of convenience and control.

  • Library Management System: A Comprehensive Solution for Managing Books, Students, and Administrative Tasks

    • The project, "Library Management System," is a comprehensive solution for managing a library's books, students, and administrative tasks. The system includes a login system where users can enter their username and password to access the system. A simple text file is used to store the usernames and passwords for both admin and students. The system has separate interfaces for the admin and student. The admin interface includes options to add, modify, and delete books, while the student interface includes options to issue and return books. One of the core features of the system is a function to issue a book. This function checks if the book is available and if the student has any outstanding fines. If the book is available and the student doesn't have any fines, the function updates the book's availability status and records the student's name and the due date for the book. Another important feature is the function to return a book. This function checks if the student has the book and updates the book's availability status. If the student returns the book after the due date, the function calculates the fine and updates the student's fine amount. The system also includes a function to change the password. This function prompts the user to enter their current password and then their new password. The function updates the password in the file storing usernames and passwords. , the system includes a function to handle the fine charging system. This function will be called when returning a book and will calculate the fine. The project is implemented using classes and objects to organize the code, making it more readable and maintainable. This allows for easy modification and expansion of the system in the future.
